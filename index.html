Habilidades

  • HTML

    Utilizo para a estruturação e organização do conteúdo das páginas web, definindo cabeçalho, corpo, seções e rodapé, com a semântica apropriada para ajudar usuários e programadores a entender toda a estrutura e garantir uma boa pontuação SEO ao site.

  • CSS

    O CSS é fundamental para que eu tenha um controle preciso da aparência visual dos elementos HTML, como cores, fontes, tamanhos de texto, etc. Também auxilia a responsividade em minhas páginas, adaptando a página para celulares, tablets e monitores.

  • JavaScript

    Ao conhecer JavaScript, posso manipular o DOM de uma página, adicionando, removendo e alterando elementos da página sem a necessidade de recarrega-la, isso permite criar diversas interações dinâmicas, como menus suspensos, animações e validação de formulários.

  • Figma

    Permite entender melhor os aspectos do design de uma interface de usuário, como a estrutura, a hierarquia de informações, as cores e as fontes, isso melhora a qualidade do meu código e reduz o tempo gasto na construção e atualização das páginas.

  • Scrum

    Ajuda a melhorar o meu desempenho em uma equipe ágil e nas sprints de desenvolvimento, trabalhando de forma auto-organizada com outros membros, auxiliando na determinação da complexidade e tempo necessário para desenvolver os itens do Product Backlog.

  • Agile

    Utilizo os quatro valores do manifesto ágil para guiar a forma como desenvolvo meus projetos, valores como “indivíduos e interações mais que processos e ferramentas”, “colaboração com o cliente mais que negociação de contratos” são fundamentais nesse meio.

  • Github

    O Git me permite compartilhar o código dos meus projetos, trabalhar em equipe e utilizar o versionamento para ter acesso a todas as modificações realizadas e participar em projetos de código aberto, ajudando outros desenvolvedores.

  • Jira

    No Jira posso melhorar a comunicação nas sprints de desenvolvimento do projeto com os demais integrantes da equipe, estabelecendo as tarefas, determinando prazos, reportando e rastreando problemas no código para que as atualizações e correções sejam feitas rapidamente.

  • UX & UI

    Conhecendo mais sobre o design de interface do usuário, posso compreender melhor os requisitos do projeto e as necessidades do usuário final, bem como melhorar a interface dos meus próprios projetos para atrair mais pessoas para conhecer o meu trabalho.
